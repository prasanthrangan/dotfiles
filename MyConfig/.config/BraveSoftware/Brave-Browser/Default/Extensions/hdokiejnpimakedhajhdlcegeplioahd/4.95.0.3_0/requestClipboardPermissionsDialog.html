Firefox has updated their security policies, so we're required to ask you to allow LastPass to "Input data to the clipboard". This allows you to copy information from LastPass, such as passwords and usernames.

  1. Click Ask me for permission. The Firefox permission request is displayed.
  2. Click Allow in the permission request.

Note:

  • If you deny permission, you won't be able to copy information from LastPass.
  • To reset this permission, reinstall the LastPass browser extension.
