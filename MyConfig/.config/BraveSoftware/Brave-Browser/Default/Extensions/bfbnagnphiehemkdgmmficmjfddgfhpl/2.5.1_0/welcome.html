# Features

This is the only extension on the market that has two major functionalities that no other extension in this category has.

  • Works on all video streaming platforms.
  • Works on all embedded videos.

# Options

The extension is fairly easy to use, but very powerful and useful.

  • Pause the extension.It will stop listening for all memory consuming events and restore any DOM manipulations.
  • Toggle between modes.Toggle between different modes that will affect the video: normal, upscale and stretch.
  • Configurable shortcuts.Set the desired keystroke, with min/max of 3 keys to control pause or mode toggling.

# Defaults

As mentioned, you can configure shortcuts to a key combination you prefer. However, after initial installation, it comes with sensible defaults.

  • Pause the extension.

    Control Shift X

  • Toggle between modes.

    Control Shift Z
